**🌟 Overview:**
A robust API for managing bookstore operations, developed using Node.js and Express.

**✨ Features:**
- 📚 Full CRUD operations for books
- 🔒 User authentication and role-based authorization

**🚀 How to Run:**
1. Clone the repository:
   ```bash
   git clone https://github.com/achovfx/nodejs-bookStoreApi
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Use tools like Postman to test API endpoints.
